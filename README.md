### Тестовые задачи

В этом задании вам предстоит создать агентов для небольшого интернет-магазина электроники (наушники, гарнитуры, клавиатуры, мыши и электронные книги). Вам предстоит выполнить три задания, каждое из которых предполагает использование новой архитектуры:

Агент, вызывающий инструменты (цикл ReAct) — модель ищет товары и управляет корзиной покупок, вызывая инструменты в итеративном цикле ReAct. Вы напишете описания функций инструмента, сгенерируете схему инструмента с помощью convert_to_openai_tool и реализуете run_shopping_agent — цикл ReAct, который отправляет сообщения в большую языковую модель, выполняет вызовы инструмента и возвращает итоговый текстовый ответ. 

Memory Agent — агент получает доступ к долговременной памяти (профиль пользователя, сохраненный на диске) и кратковременной памяти (история диалогов), что позволяет ему запоминать предпочтения между сеансами и результаты поиска в рамках одного сеанса. Вам нужно будет реализовать функции load_profile / save_profile для чтения и записи профиля в формате JSON, определить схему инструмента update_profile и реализовать функцию run_memory_agent, которая подгружает профиль в системную подсказку и сохраняет историю диалогов между ходами. 

 Многоагентная система — четыре специализированных агента: Retriever, Pros Analyst, Cons Analyst и Ranker — координируются Orchestrator, который распределяет работу, собирает результаты и выдает итоговую рекомендацию с указанием плюсов и минусов. Вам нужно реализовать все четыре класса агентов и CoordinatorAgent, который запускает их по очереди, передавая данные через общий AgentContext.
